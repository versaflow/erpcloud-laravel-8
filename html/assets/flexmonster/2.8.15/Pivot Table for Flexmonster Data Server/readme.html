How to connect to CSV/JSON files and relational databases
with Flexmonster Data Server

Flexmonster Data Server is a special server-side tool that implements the custom data source API. It fetches data from a data source, aggregates it, and then passes the data to Flexmonster Pivot in a ready-to-show format. The Data Server significantly reduces the time of data loading and allows analyzing large datasets.

Flexmonster Data Server supports the following data sources:

  • CSV and JSON files
  • MySQL databases
  • Microsoft SQL Server databases
  • Microsoft Azure SQL databases
  • PostgreSQL databases
  • Oracle databases
