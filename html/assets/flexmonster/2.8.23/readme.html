Thank you for downloading Flexmonster Pivot Table & Charts!

See advantages of Flexmonster Pivot

  • Fast & powerful. Flexmonster was designed to do more in less time. It visualizes large datasets from JSON/CSV files, different databases, and OLAP cubes at a moment's notice. The maximum data size is limited only by your device capacity.
  • Easily customizable. The component's extensive JavaScript API allows adjusting the functionality to your needs, and its open-source CSS styles can be used as a template for custom themes.
  • Offers advanced reporting features. Your users can analyze data with different UI options such as filtering, sorting, conditional formatting, and many more.

Check out supported data sources

Flexmonster Pivot supports the most common data sources:

  • CSV and JSON files
  • SQL databases: MySQL, Microsoft SQL Server, and others
  • MongoDB
  • Microsoft Analysis Services OLAP cubes
  • Elasticsearch
  • Any data source with the custom data source API
